# InvestPilot

一个面向个人投资者的股票投资管理网站首版。

## 当前能力

- 展示个人持仓、仓位市值、累计盈亏和个股逻辑
- 展示 AI 风险信号、事件流和组合摘要
- 预留财联社、东方财富、同花顺的数据接入层
- 明确区分可优先接入、规划接入和受限接入的数据源状态

## 本地启动

```bash
npm install
npm run dev
```

默认启动地址：

`http://localhost:5173`

## 当前数据源策略

- 同花顺：优先作为正式行情与财务数据接入对象
- 东方财富：适合作为专业数据补充层，建议通过官方商业数据产品接入
- 财联社：建议通过官方授权资讯合作方式接入，首版不要直接依赖网页抓取
